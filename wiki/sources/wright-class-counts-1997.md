# Class Counts (Wright, 1997)

Erik Olin Wright's synthesis of the multi-country **Comparative Class Analysis Project**: neo-Marxist class maps (exploitation + authority + skill) applied to comparative distributions, boundary permeability, class–gender interaction, and class consciousness. Four themes structure the book: class structure and its transformation; permeability of class boundaries; class and gender; class consciousness.
