# Class Counts — Deep Analysis

> Studies Hub deep analysis. Summary: [[wright-class-counts-study]]. Source: [[wright-class-counts-1997]]. All findings **attributed** to Wright / project data (mostly 1980s surveys).

**Analysis standard:** design table · findings table · load-bearing passages · afterlife · nine sections. Reconstruct the project as conducted — a multi-decade survey program folded into a book, not a single fieldwork campaign.

## Question and Intellectual Context

*Class Counts* asks whether a reconstructed, exploitation-centered Marxist class concept can organize comparative knowledge about advanced capitalism once operationalized for survey research. Four themes: class structure and transformation; boundary permeability; class and gender; class consciousness. The book sits in the Cambridge **[[analytical-marxism|Analytical Marxism]]** series (Cohen–Elster–Roemer): Marxism as concepts and questions, not as a method that bans quantification.

Wright's dual audience (conclusion): show non-Marxists that class still structures outcomes; show Marxists that systematic quantitative methods can serve radical theory. Foils include classical proletarianization (Braverman), post-industrial optimism, Goldthorpe family-class assignment, Weberian life-chance class, and feminist critiques of class primacy.

## Design and Methods as Conducted

| Question | Unit | Sample / Site | Method | Period |
|---|---|---|---|---|
| How are class locations distributed cross-nationally? | Individual job location | Multi-country labor-force surveys | Contingency tables; 12-location map | c. 1980–87 |
| How did the US class structure change? | Class × sector | US census + project estimates | Shift-share; decade comparisons | 1960–90 |
| Did self-employment revive? | Self-employed status | US time series + sectoral decomposition | Time series; shift-share | mid-20th c.–1990 |
| How permeable are class boundaries? | Dyadic events (mobility, friendship, marriage) | Men/women matrices by country | Loglinear quasi-independence models | project surveys |
| How do class and gender interact? | Individual + household | Dual-earner subsets; US/Sweden | Identity models; housework Tobit | project surveys |
| How does location map to consciousness? | Individual ideology scores | Sweden, US, Japan | Coalition maps; micro multivariate models | project surveys |

**Design as conducted (not as idealized).** NSF-originated US–Italy plan expanded into a ~15-country network after 1978 Uppsala lectures; Italian partner Luca Perrone died 1981 before Italian fielding. Wright became unintended coordinating node. Articles written 1987–95 retained slightly inconsistent operationalizations — "foolish consistency" deliberately not enforced in the book. Samples are **labor-force / housewife** frames: no serious portrait of the ruling class or the underclass; race underpowered for deep US race–class analysis.

## The Site / Sample / Data

The "site" is a **network of national surveys** sharing a class module (property, authority, skill) with country-specific fieldwork. Core countries for most analyses: United States, Sweden, Norway, Canada, Japan (Australia appears in gender chapters). Wright repeatedly stresses **operational underdetermination**: absolute class-size percentages are threshold-sensitive; comparative *differences* are more robust.

### Load-bearing theoretical findings (attributed)

**Exploitation (three criteria).** Antagonistic dependence of exploiters' welfare on the exploited; asymmetrical exclusion from productive resources; appropriation of the fruits of labor — distinguishing exploitation from mere economic oppression (e.g., some underclass configurations).

**Contradictory locations.** Managers and experts occupy "privileged appropriation locations within exploitation relations" (loyalty rents; skill rents) while remaining employees — neither pure capital nor pure labor.

**Permeability surprise.** Marxist expectation that property is hardest to cross is confirmed; ranking of authority vs skill as more open was less theoretically settled — data put **authority most permeable**.

**Sites of class formation.** Point-of-production thesis underspecified: Swedish class identity tracks own job more; US married women's identity more household/consumption-mediated — class formation sites vary.

## Analytic and Rhetorical Moves

**1. The shmoo parable (Ch. 1).** Li'l Abner creatures that supply necessities without toil threaten both capital (labor recruitment) and patriarchal dependence — a pedagogical device (via G. A. Cohen) for antagonistic material interests without equations first.

**2. Independent-variable specialty.** Class analysis brackets other causes not because they are unreal but because the research program isolates class as IV — a methodological, not metaphysical, primacy claim (defended against feminist primacy critiques in Ch. 9).

**3. Confirmations / surprises / reconstructions (Ch. 17).** Wright refuses both dogmatism (ignore surprises) and abandonment (throw out the map): theory is revised in light of results while retaining exploitation-centered structure.

**4. Dual agenda close.** Quantitative methods for Marxists; class for non-Marxists — the book's political-intellectual stance.

## Reception and Theoretical Impact

Standard citation for multi-dimensional neo-Marxist class schemes; influences stratification, political sociology of class voting/consciousness, and gender–class research. Pairs with *Classes* (1985) as theory + evidence. Later Wright "real utopias" work shifts normative focus; *Class Counts* remains the empirical peak of the survey program.

## Critiques, Restudies, and Replications

- **Operational arbitrariness** — Wright admits cut-points; Marshall et al. and others debate reliability of authority/skill measures.
- **Missing poles** — no ruling class, thin underclass, thin race (author's own cautions).
- **Death of class / individualization** — rival literatures (Beck, etc.) not fully engaged as project-internal tests; record as external debate ([[individualization-death-of-class-debate]]).
- **Feminist autonomy** — housework noneffects support gender-autonomy hypotheses that undercut strong class-determination of domestic labor.
- **Replication status unknown** in this vault for full multi-chapter package; individual articles had peer review; independent replications of specific coefficients not ingested.

## Current Standing

A classic of comparative class analysis. Treat coefficients as **1980s–90s advanced-capitalist survey results**, not timeless laws. Conceptual apparatus (contradictory locations, permeability, mediated locations) remains live vocabulary.
