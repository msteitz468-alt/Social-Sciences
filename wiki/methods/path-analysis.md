# Path Analysis

A system of standardized regression equations representing hypothesized direct and indirect effects among ordered variables. In sociology, the canonical application is the Blau–Duncan stratification model in *[[blau-duncan-american-occupational-structure-1967|The American Occupational Structure]]* (1967).

## What It Does

Given a recursive causal diagram, path analysis estimates path coefficients that partition correlations into direct paths and indirect paths through intervening variables (e.g. origin → education → occupation).

## Procedure

Specify causal order → write structural equations → estimate standardized coefficients (often via OLS) → interpret direct/indirect effects and residual paths. Blau and Duncan devote Chapter 5 to the basic model and path coefficients for OCG men.

## Assumptions and Limitations

Correct ordering; linearity; no reciprocal causation in the simple recursive form; measurement of status (SEI) shapes results. Omitted variables (ability, networks, labor demand) bias paths.

## History

Sewall Wright's path coefficients in genetics; mid-1960s sociological adoption. Blau & Duncan 1967 made the method a standard tool of stratification research.

## Exemplary Applications

[[blau-duncan-american-occupational-structure-study]]; later status-attainment elaborations.

## Debates

Causal identification from cross-sectional survey data; path models vs log-linear mobility models; continuous status vs class categories.
