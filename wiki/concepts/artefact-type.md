# Artefact-Type

Level in [[clarke-david-l|Clarke]]'s hierarchical model (*[[clarke-analytical-archaeology-1978|Analytical Archaeology]]*): a recurrent polythetic configuration of attributes defining a type within artefact populations, treated as a system with phase, time, and system pattern regularities.

## Definition as Coined

*(Attributed — Clarke, ch. 5 and Definitions.)*

> "an artefact-type is — an homogeneous population of artefacts which share a consistently recurrent range of attribute states within a given polythetic set."

Glossary: intermediate-level affinity, perhaps 30–60%, uniting the population as a whole. Nested type ranks:

| Rank | Affinity (indicative) | Content |
|---|---|---|
| Type group / family | low, perhaps ≤30% | Affinally related collateral types sharing a functional-material attribute subset |
| Specific type | intermediate, ~30–60% | Polythetic homogeneous population |
| Subtype / variant | high, ~60–90% | Homogeneous subpopulation within a type's polythetic set |

## Semantic History

"Type" is among archaeology's oldest tools ([[typological-method]]). Clarke re-specified it as a **population system** inside a polythetic hierarchy, rejecting monothetic exclusive attribute lists and the fiction that a single "mean" or "typical" specimen represents the type (artefacts mix archaic, modal, and prototypical attribute states).

## Operationalizations

- **Attribute constellations:** core tightly covarying clusters; essential attributes at moderate co-occurrence; sporadic penumbra.
- **Phase regularities:** frequency distributions and modal states within a time-slice.
- **Time regularities:** lenticular type-frequency curves; formative/mature/disjunctive ontogeny patterns; staggered attribute-threshold nodes defining transform types.
- **System regularities:** type behaviour under sociocultural/environmental dislocation.
- **Numerical taxonomy:** phenon lines on dendrograms for more objective rank comparison (Sokal & Sneath).

Illustrations in the book include Beaker pottery attributes, Hallstatt swords, Iron Age fibulae, Acheulean handaxes, Mariana pottery frequencies, Aurignacian seriations.

## Applications

- Bridge between raw artefacts and [[archaeological-assemblage|assemblages]]/[[archaeological-culture|cultures]].
- Quantitative typology and seriation under [[processual-archaeology|New Archaeology]].

## Critiques and Limitations

- Affinity bands are heuristic.
- Attribute choice remains theory-laden.
- Type-as-population still often collapsed to type-as-ideal-specimen in practice.
- Does not decide social meaning of types (style vs function debates live elsewhere — e.g. Binford–Bordes).
