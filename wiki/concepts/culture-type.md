# Culture Type

## Definition as Coined

[[steward-julian-h|Steward]] (*[[steward-theory-of-culture-change-1955|Theory of Culture Change]]*, Ch. 1 and Ch. 5): a **culture type** (for determining cross-cultural parallels and regularities) differs from a [[culture-area]]:

1. Characterized by **selected features** rather than total element content.
2. Selection of diagnostic features determined by the **problem and frame of reference**.
3. Selected features presumed to have the **same functional interrelationship** in each case.

Cross-cultural types consist of the [[cultural-core|cultural cores]] of societies that share similar ecological adaptations and similar [[levels-of-sociocultural-integration]], even across different areas. Steward contrasted **uniformities** (historical connection / diffusion within areas) with **regularities** (independent functional parallels).

## Semantic History

Related to wider anthropological/archaeological typology; Steward's usage is specifically ecological-evolutionary. Distinct from archaeological "culture" as material assemblage (wiki `cultures/` layer). Illustrations Steward named: Wittfogel's irrigation/"oriental absolute" society; [[patrilineal-band]]; Redfield's folk society (which Steward also re-read as a *level*, not only a type); feudal society (Japan and Europe); predatory band as post-contact type by form/function.

## Operationalizations

Core + integration level; comparative cases of the same type in multiple areas; developmental types for irrigation cradles (Formative, Florescence, Cyclical Conquests as organizational eras, not metal ages).

## Applications

Multilinear comparison; Aboriginal America methodological discussion (Ch. 5); substantive chapters 6–12 of *Theory of Culture Change*.

## Critiques and Limitations

Type boundaries analytic; danger of cherry-picking core traits; Galton's problem if types are not independent; "hunting and gathering" as a single type rejected by Steward himself as too coarse.
