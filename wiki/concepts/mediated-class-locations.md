# Mediated Class Locations

## Definition as Coined

*(Attributed — Wright, *[[wright-class-counts-1997|Class Counts]]*.)* A **mediated class location** is a class position shaped not only by one's own direct relation to production (job) but by **class relations mediated through the household** — especially the spouse's class location. The concept answers: what is the class location of a married person (classically: married women) when household members occupy different class locations?

Wright rejects reducing class to either pure individual job *or* pure household head: both **direct** and **mediated** locations can shape interests and consciousness, with relative weight varying by political-economic context (e.g., production-centered class formation in Sweden vs consumption/neighborhood-centered class talk in the US — as reconstructed in Ch. 17).

## Semantic History

Part of the class-and-gender program responding to feminist critiques of class analysis. Related to but distinct from "family class" assignment by male head alone.

## Operationalizations

Dual-earner samples; models of class identity and attitudes with own-class and spouse-class predictors; country comparisons (esp. Sweden vs US).

## Applications

Cross-class families (Ch. 8); married women's class identity (Ch. 10); consciousness models (Chs. 14–17).

## Critiques and Limitations

Sample restricted to labor force (limits household universe); dual-earner focus; gender ideology and unpaid labor only partly captured. Attributed.
